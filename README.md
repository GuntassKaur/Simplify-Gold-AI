# Simplify Gold AI

An AI-powered Digital Gold Investment Assistant inspired by Simplify Money's KuberAI experience.

## Features

* AI-powered gold investment guidance
* Digital Gold recommendation engine
* Live Gold Price Integration
* Real-time USD to INR conversion
* User Registration and Management
* Transaction History Tracking
* Portfolio Dashboard
* Unique Transaction ID Generation
* SQLite Database Integration
* FastAPI Backend
* Cloud Deployment on Render

## Tech Stack

### Backend

* Python
* FastAPI
* SQLAlchemy
* SQLite

### AI

* Gemini API

### Frontend

* HTML
* CSS
* JavaScript

### Deployment

* GitHub
* Render

## Run Locally

```bash
git clone https://github.com/GuntassKaur/Simplify-Gold-AI.git

cd Simplify-Gold-AI

pip install -r requirements.txt

uvicorn app.main:app --reload
```
